Documentation and the GamePlay video  has been embedded in the game's main website.
To Fullly Enjoy  the game in the multiplayer mode a touch computer screen is recommended! The game can be played using a mouse by using pluralinput's software.
Plural input enables the user to use multiple mouse inputs in one PC. The software can be downloaded from https://pluralinput.com/. 

Fruit and Frags was tested ona Microsoft surface laptop, which has a multitouch touchscreen.

To run the game:
1) Install node.js , using npm install in the parent directory 
2) point the web browser to http://localhost:8080/
3) Enjoy
