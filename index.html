Description

Inspired by the very popular mobile game Fruit Ninja, which is available on only Android and IOS devices. I created Fruits and Frags using javaScript, this will enable people to play the game on any computer, using just the webbrowser. In Fruit and Frags, the player slices fruit with a blade controlled via the mouse or using touches. As the fruit is thrown onto the screen, the player swipes across the screen to create a slicing motion, attempting to slice the fruit in half. Players must slice all fruit; if three fruits are missed,the game ends. Bombs are occasionally thrown onto the screen, and will also end the game should the player slice them.

Features list

  • Local multiplayer on a single device
  • 2D Collision detection
  • Local database using a JSON file
  • Interactive buttons
  • OOP based design
  • Dynamic 2D arcade physics
  • Animations of fruits, UI and a slice
  • Cool game sounds
